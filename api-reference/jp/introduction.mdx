# API リファレンス - はじめに

MoEngage API を使用して、プラットフォームの機能をプログラムで操作できます。

## 利用可能なAPI

- **プロダクトカタログAPI**: 商品カタログの管理
- **キャンペーンAPI**: マーケティングキャンペーンの管理
- **グローバルスロットリングAPI**: レート制限の管理
- **プッシュテンプレートAPI**: プッシュ通知テンプレートの管理

## 認証

APIを使用するには、適切な認証トークンが必要です。
