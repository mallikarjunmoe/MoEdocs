# Navigate to Your Account's Login Page

Accessing your MoEngage account login page is straightforward. This guide will help you find the correct login URL and navigate to your account.

## Finding Your Login Page

### Standard Login URL
The main MoEngage login page is available at:
- **URL**: [https://dashboard.moengage.com/v4](https://dashboard.moengage.com/v4)
- **Alternative**: [https://app.moengage.com](https://app.moengage.com)

### From MoEngage Website
1. Visit [www.moengage.com](https://www.moengage.com)
2. Click the "Login" button in the top navigation
3. You'll be redirected to the login page

### Bookmarking the Login Page
- **Browser Bookmark**: Save the login URL for quick access
- **Mobile Bookmark**: Add to your mobile device's home screen
- **Password Manager**: Save the URL in your password manager

## Login Page Features

### Login Form
- **Email Field**: Enter your registered email address
- **Password Field**: Enter your account password
- **Remember Me**: Option to stay logged in
- **Sign In Button**: Submit your credentials

### Additional Options
- **Forgot Password**: Link to reset your password
- **Sign Up**: Link to create a new account
- **Help**: Access to support and documentation

### Security Features
- **SSL Encryption**: Secure connection for login
- **CAPTCHA**: Protection against automated attacks
- **Rate Limiting**: Protection against brute force attacks

## Account-Specific Login

### Custom Login URLs
Some organizations may have custom login URLs:
- **Enterprise Accounts**: May use custom domains
- **SSO Integration**: May redirect to company login
- **White-label Solutions**: May use branded login pages

### Finding Your Custom URL
1. **Check Email**: Look for login links in MoEngage emails
2. **Contact Admin**: Ask your workspace administrator
3. **Support**: Contact MoEngage support for assistance

## Mobile Access

### Mobile Browser
1. Open your mobile browser
2. Navigate to the login URL
3. Enter your credentials
4. Enable "Remember Me" for convenience

### Mobile App
- **iOS**: Download from the App Store
- **Android**: Download from Google Play Store
- **Features**: Full dashboard access on mobile devices

## Troubleshooting Access Issues

### Common Problems

#### Page Not Loading
- **Check Internet Connection**: Ensure you have a stable connection
- **Try Different Browser**: Switch to a different web browser
- **Clear Cache**: Clear your browser cache and cookies
- **Disable Extensions**: Temporarily disable browser extensions

#### Wrong Login Page
- **Verify URL**: Double-check the login URL
- **Check Bookmarks**: Ensure your bookmark is correct
- **Search Engine**: Use a search engine to find the official login page

#### Access Denied
- **Check Credentials**: Verify your email and password
- **Account Status**: Ensure your account is active
- **Contact Support**: Reach out to MoEngage support

### Browser Compatibility
- **Supported Browsers**: Chrome, Firefox, Safari, Edge
- **Version Requirements**: Use recent browser versions
- **JavaScript**: Ensure JavaScript is enabled
- **Cookies**: Allow cookies for the MoEngage domain

## Security Best Practices

### Safe Login Practices
- **Direct URL**: Always type the URL directly or use bookmarks
- **HTTPS**: Ensure the connection is secure (https://)
- **Public Wi-Fi**: Avoid logging in on public networks
- **Shared Computers**: Log out when using shared devices

### Password Security
- **Strong Passwords**: Use complex, unique passwords
- **Password Manager**: Use a password manager for security
- **Regular Updates**: Change passwords periodically
- **Two-Factor Authentication**: Enable 2FA for extra security
