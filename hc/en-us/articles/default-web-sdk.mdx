# Default Web SDK

The MoEngage Web SDK provides comprehensive customer engagement features for web applications, enabling push notifications, in-app messaging, and user tracking.

## Overview

### What is the Web SDK?
The MoEngage Web SDK is a JavaScript library that enables:
- **Web Push Notifications**: Send and receive push notifications in browsers
- **In-App Messaging**: Display in-app messages and campaigns
- **User Tracking**: Track user behavior and events
- **Analytics**: Collect and analyze user data
- **Personalization**: Deliver personalized experiences

### Key Features
- **Cross-Browser Support**: Works on all modern browsers
- **Service Worker**: Uses service workers for push notifications
- **Rich Notifications**: Support for rich media notifications
- **Deep Linking**: Handle deep links and navigation
- **Privacy Compliant**: GDPR and privacy compliant

## Installation

### CDN Integration
Include the SDK via CDN:

```html
<script src="https://cdn.moengage.com/webpush/moe_webSdk.min.latest.js"></script>
```

### NPM Installation
Install via npm:

```bash
npm install moengage-web-sdk
```

### Manual Download
Download the SDK file and include it in your project:

```html
<script src="path/to/moe_webSdk.min.js"></script>
```

## Configuration

### Step 1: Initialize SDK
Initialize the SDK with your App ID:

```javascript
MoEngage.initialize({
    appId: 'YOUR_APP_ID',
    debug: true // Enable for development
});
```

### Step 2: Configure Push Notifications
Set up push notifications:

```javascript
MoEngage.initialize({
    appId: 'YOUR_APP_ID',
    push: {
        enabled: true,
        vapidKey: 'YOUR_VAPID_KEY'
    }
});
```

### Step 3: Set Up Service Worker
Create a service worker file (`sw.js`):

```javascript
importScripts('https://cdn.moengage.com/webpush/moe_webSdk.min.latest.js');

MoEngage.initialize({
    appId: 'YOUR_APP_ID'
});
```

## Basic Usage

### User Identification
Identify users with unique IDs:

```javascript
MoEngage.setUserId('user123');
MoEngage.setUserAttribute('email', 'user@example.com');
MoEngage.setUserAttribute('name', 'John Doe');
```

### Event Tracking
Track user events:

```javascript
MoEngage.trackEvent('Purchase', {
    amount: 99.99,
    currency: 'USD',
    product_id: 'product123'
});
```

### Push Notifications
Request push notification permission:

```javascript
MoEngage.requestPushPermission()
    .then(() => {
        console.log('Push permission granted');
    })
    .catch((error) => {
        console.log('Push permission denied:', error);
    });
```

## Advanced Features

### In-App Messaging
Display in-app messages:

```javascript
MoEngage.showInApp();
```

### Deep Linking
Handle deep links:

```javascript
MoEngage.onDeepLink((url) => {
    console.log('Deep link received:', url);
    // Handle the deep link
});
```

### User Attributes
Set user attributes:

```javascript
MoEngage.setUserAttribute('age', 25);
MoEngage.setUserAttribute('location', 'New York');
MoEngage.setUserAttribute('subscription', 'premium');
```

## Customization

### Notification Styling
Customize notification appearance:

```javascript
MoEngage.initialize({
    appId: 'YOUR_APP_ID',
    push: {
        enabled: true,
        icon: 'path/to/icon.png',
        badge: 'path/to/badge.png'
    }
});
```

### Event Properties
Add custom properties to events:

```javascript
MoEngage.trackEvent('Product View', {
    category: 'electronics',
    brand: 'Samsung',
    price: 299.99
});
```
