### Add Maven Repository

Add *mavenCentral()* repository in the project-level *build.gradle* file.

```bash Groovy lines
buildscript {
    repositories {
        mavenCentral()
    }
}
allprojects {
    repositories {
        mavenCentral()
    }
}
```

### Enable Java 8

The SDK is target and source compatible with version 8 of the Java Programming Language. Enable Java 8 in the application *build.gradle(.kts)* if not done already.

<Tabs>
  <Tab title="build.gradle">
    ```groovy lines
    android {
      ...
      compileOptions {
         sourceCompatibility JavaVersion.VERSION_1_8
         targetCompatibility JavaVersion.VERSION_1_8
      }
      kotlinOptions {
         jvmTarget = '1.8'
      }
    }
    ```
  </Tab>
  <Tab title="build.gradle.kts">
    ```kotlin lines
    android {
      ...
      compileOptions {
         sourceCompatibility = JavaVersion.VERSION_1_8
         targetCompatibility = JavaVersion.VERSION_1_8
      }
      kotlinOptions {
         jvmTarget = "1.8"
      }
    }
    ```
  </Tab>
</Tabs>

For more information about samples, refer to [Android Sample](https://github.com/moengage/Android-Sample).
