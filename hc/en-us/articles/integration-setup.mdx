# Integration Setup

Setting up MoEngage integration involves configuring your application to work with MoEngage's platform, enabling you to track user behavior, send campaigns, and analyze performance.

## Overview

### What is Integration Setup?
Integration setup is the process of:
- **Configuring Your App**: Setting up your application to work with MoEngage
- **Connecting Systems**: Establishing connections between your app and MoEngage
- **Testing Integration**: Verifying that everything works correctly
- **Optimizing Performance**: Ensuring optimal performance and reliability

### Key Components
- **SDK Integration**: Installing and configuring the MoEngage SDK
- **API Configuration**: Setting up API connections and authentication
- **Data Mapping**: Mapping your data to MoEngage's data structure
- **Testing**: Testing the integration to ensure it works properly

## Prerequisites

### Before You Start
- **MoEngage Account**: You need an active MoEngage account
- **App ID**: Your MoEngage App ID
- **API Keys**: Your MoEngage API keys
- **Development Environment**: A development environment for testing

### Required Information
- **App Details**: Your app name, bundle ID, and platform
- **Data Center**: Your preferred data center region
- **Tracking Requirements**: What data you want to track
- **Campaign Requirements**: What campaigns you want to send

## Step-by-Step Setup

### Step 1: Create MoEngage Account
1. **Sign Up**: Create a MoEngage account if you don't have one
2. **Verify Email**: Verify your email address
3. **Complete Profile**: Complete your profile information
4. **Choose Plan**: Select a plan that meets your needs

### Step 2: Create App in MoEngage
1. **Log In**: Log into your MoEngage dashboard
2. **Create App**: Create a new app in MoEngage
3. **Configure Settings**: Set up basic app settings
4. **Get Credentials**: Note down your App ID and API keys

### Step 3: Install SDK
1. **Choose Platform**: Select your platform (iOS, Android, Web)
2. **Download SDK**: Download the appropriate SDK
3. **Install SDK**: Install the SDK in your application
4. **Configure SDK**: Configure the SDK with your credentials

### Step 4: Configure Tracking
1. **Set Up Events**: Configure event tracking
2. **Set Up User Attributes**: Configure user attribute tracking
3. **Set Up Campaigns**: Configure campaign settings
4. **Test Tracking**: Test that tracking is working

## Platform-Specific Setup

### iOS Setup
1. **Install CocoaPods**: Install CocoaPods if not already installed
2. **Add to Podfile**: Add MoEngage SDK to your Podfile
3. **Run pod install**: Run `pod install` to install dependencies
4. **Configure in AppDelegate**: Configure SDK in your AppDelegate

### Android Setup
1. **Add to build.gradle**: Add MoEngage SDK to your build.gradle
2. **Sync Project**: Sync your project to download dependencies
3. **Configure in Application**: Configure SDK in your Application class
4. **Set Up Permissions**: Set up necessary permissions

### Web Setup
1. **Include SDK**: Include the MoEngage Web SDK in your HTML
2. **Initialize SDK**: Initialize the SDK with your credentials
3. **Set Up Service Worker**: Set up service worker for push notifications
4. **Configure Tracking**: Configure event and user tracking
