# ビルド設定の構成

Android SDK のビルド設定を構成する方法について説明します。

## 必要な設定

- Gradle設定
- 依存関係の追加
- プロガード設定
- マニフェスト設定

## 設定手順

1. `build.gradle` ファイルを編集
2. 必要な依存関係を追加
3. プロガードルールを設定
4. アプリケーションをビルド
