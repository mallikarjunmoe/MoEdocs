# SDKバージョン9.23.0未満でのユニークID設定

SDKバージョン9.23.0未満でユニークIDを設定する方法について説明します。

## ユニークIDとは

ユニークIDは、ユーザーを一意に識別するためのIDです。

## 設定方法

### iOS
```swift
// ユニークIDの設定例
MoEngage.sharedInstance.setUniqueId("user_unique_id")
```

### Android
```java
// ユニークIDの設定例
MoEngage.getInstance().setUniqueId("user_unique_id");
```
