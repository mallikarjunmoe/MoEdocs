# SDK統合

MoEngage SDK をアプリケーションに統合する方法について説明します。

## 前提条件

- iOS 12.0 以上
- Xcode 12.0 以上

## インストール手順

1. CocoaPods を使用してSDKをインストール
2. プロジェクトにSDKを追加
3. 初期設定を行う
