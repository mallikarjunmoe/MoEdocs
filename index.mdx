## Authentication

All MoEngage APIs are authenticated using a combination of Basic Authentication and a custom MOE-APPKEY header.

You can find your credentials in the MoEngage Dashboard under Settings > Account > APIs. For detailed examples, please see the authentication section within each API endpoint.

## Quick Start

<Steps>
  <Step title="Get Your API Credentials">
    Log into your [MoEngage Dashboard](https://dashboard.moengage.com/v4) and navigate to Settings > Account > APIs to retrieve your API key and secret.
  </Step>
  <Step title="Choose Your API">
    Browse our comprehensive API collection above to find the endpoints that match your integration needs.
  </Step>
  <Step title="Start Building">
    Use our interactive API documentation to test endpoints and integrate MoEngage into your application.
  </Step>
</Steps>
