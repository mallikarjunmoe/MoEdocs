## Authentication

All MoEngage APIs are authenticated using a combination of Basic Authentication and a custom MOE-APPKEY header.

You can find your credentials in the MoEngage Dashboard under Settings > Account > APIs. For detailed examples, please see the authentication section within each API endpoint.
